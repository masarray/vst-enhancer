# ArSonKuPik VST — Free Public Beta

**ArSonKuPik v0.5.x** is a free public beta for Windows VST3 and Standalone. No activation, login, audio watermark, or expiry is required for the beta build.

ArSonKuPik is a smart audio enhancer for mastering, mix bus, tracks, and creative processing. The DSP and application source remain private in `masarray/askp-vst`; this public repository contains the landing site, documentation, release metadata, and downloadable Windows binaries.

## Public beta status

- Free to test in personal and commercial audio-production projects
- No activation required
- No automatic expiry
- Windows 10/11 x64
- VST3 and Standalone application
- Unsigned build; Windows SmartScreen may show a warning

The public beta is intended to validate DAW compatibility, plugin scanning, CPU use, preset behaviour, sound quality, and stability before the first commercial stable release. Future stable versions may use a different licence model; existing beta downloads are not remotely disabled.
