# ArSonKuPik VST

Public distribution repository for **ArSonKuPik**, a smart Windows VST3 and standalone audio enhancer for mastering, mix bus, tracks, and creative processing.

<p align="center">
  <img src="site/assets/arsonkupik-vst-ui-v050.webp" width="1082" alt="ArSonKuPik VST interface showing the Mas Ari Signature preset, smart enhancement controls, A/B comparison, bypass, protect, and input-output meters">
</p>

The DSP and application source remain private in `masarray/askp-vst`. This public repository contains only:

- GitHub Pages landing site
- Public documentation
- Release metadata
- Windows installer, VST3 ZIP, Standalone ZIP, and SHA-256 checksums in GitHub Releases

## Supported platform

- Windows 10/11 x64
- VST3
- Standalone application

macOS and VST2 are intentionally not distributed.

## Download

- [Latest GitHub Release](https://github.com/masarray/vst-enhancer/releases/latest)
- [Public landing page](https://masarray.github.io/vst-enhancer/)

## Unsigned preview

Current Windows packages are unsigned. Windows SmartScreen may display a warning. Verify every download against `SHA256SUMS.txt` from the same GitHub Release.
